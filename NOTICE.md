## Attribution

This repository references the upstream project below:

- [78/xiaozhi-esp32](https://github.com/78/xiaozhi-esp32)

The upstream project publicly describes support for device-side MCP control over capabilities such as speaker, LED, servo, GPIO, and screen-related operations.

## Scope Boundary

This repository is not a mirror of the upstream firmware and does not include private local modifications.

It only contains:

- original documentation
- protocol schemas
- clean-room example code
- a small runnable bridge translator

No private endpoints, secrets, or unpublished firmware files are included.
