# Upstream Reference

## Confirmed Upstream

- upstream repository: [78/xiaozhi-esp32](https://github.com/78/xiaozhi-esp32)
- observed license: `MIT`
- upstream description: `An MCP-based chatbot`

## Why This Upstream Matters

The upstream repository documents a device-side MCP architecture and lists control targets such as speaker, LED, servo, GPIO, and screen features. That makes it a suitable reference point for public discussion of AI-agent-to-device control patterns on ESP32.

## What Is Original In This Repo

This repository contributes original public-safe material:

- a simplified tool catalog
- a clean-room bridge translator
- command and status schemas
- clean-room ESP32 routing examples

## What Is Intentionally Not Published

- private xiaozhi branch code
- local self-host service addresses
- credentials or account-pool logic
- unpublished firmware files
